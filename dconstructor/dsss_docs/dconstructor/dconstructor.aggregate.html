dconstructor.aggregate

Builders for aggregate types: arrays and dictionaries.

class GlobalListBuilder (TBuilder,TList): AbstractBuilder!(TBuilder,TList[]);
Provides an array of the given type. Members of the array are provided statically. This array is provided for every type that requires an array of that type.

class GlobalDictionaryBuilder (TBuilder,TKey,TValue): AbstractBuilder!(TBuilder,TValue[TKey]);
Provides a static, global associative array with the given keys and values. Anything taking a dictionary of that type will be given this dictionary. The exception, of course, being when it's wrapped in a MultiBuilder.
